# Étude de marché avec ACP et clustering

## Contexte et objectifs

Ce projet consiste à analyser un marché international en appliquant une Analyse en Composantes Principales puis un clustering afin de segmenter les pays selon leurs caractéristiques.

## Soft Skills
- Interprétation de résultats statistiques avancés  
- Analyse critique  
- Rédaction d’une synthèse professionnelle  
- Vulgarisation des méthodes employées  

## Compétences
- Réalisation d’une ACP  
- Clustering (k-means, silhouette, elbow)  
- Normalisation du dataset  
- Analyse des groupes et recommandations  

## Outils utilisés
- Python  
- Pandas, Scikit-learn, Matplotlib, Seaborn
